The Prompt Trap

Everyone's first instinct: write a better prompt. But watch what actually happens.

What you askWhat actually happens
"Test the login flow"Doesn't know which app, which screen, what credentials
"Click the search button"3 elements match "search" — picks the wrong one
"Verify order was placed"Says "done" while still on the cart page
"Scroll to the product"Scrolls endlessly — no idea when to stop

A prompt without context is a test without assertions — it runs, it just doesn't mean anything.
